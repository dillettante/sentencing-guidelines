# 양형기준 적용 스킬 (Korean Sentencing Guidelines Skill)

대한민국 **양형위원회 『2026 양형기준』(2026. 7. 1. 시행)** 을 조회·적용해, 특정 범죄사실에 대한 **권고 형량범위·집행유예 판단·양형 검토 노트**를 산출하는 에이전트 스킬입니다. 양형기준이 설정된 **49개 전 범죄군**의 형량범위표·양형인자표·집행유예표·다수범죄 특칙이 구조화되어 있습니다.

요약 도구가 아니라 판단 파이프라인입니다:
`사실관계 → 범죄군·유형 특정 → 형량범위 조회 → 양형인자 평가 → 집행유예 판단 → 검토 노트`

---

## 구성

```
sentencing-guidelines/
├── SKILL.md            # 공통 엔진: 적용 5단계 + 평가·공통원칙 + 다수범죄 + 산출 포맷 + 정확성 게이트 + 49개 색인
└── references/         # 49개 범죄군 데이터 (범죄군별 1파일)
    ├── gangdo.md       # 강도
    ├── sagi.md         # 사기
    ├── hoengnyeong.md  # 횡령·배임
    └── ... (총 49개)
```

각 reference 파일: **유형정의(구성요건·적용법조) · 형량범위표(감경/기본/가중) · 양형인자표(특별·일반 × 감경·가중 × 행위·행위자) · 인자정의 · 집행유예 기준**. 전 범죄군에 공통인 평가원칙·공통원칙·표준 다수범죄 처리기준은 중복을 피해 `SKILL.md`에만 두었고, 범죄군 고유 특칙(예: 이득액 합산)만 각 파일에 있습니다.

## 사용법

에이전트 스킬 디렉터리에 배치하면 트리거 시 자동 로드됩니다.
- **Claude Code**: `~/.claude/skills/sentencing-guidelines/`
- **GitHub Copilot CLI / Amp** 등: 각 에이전트의 skills 루트에 배치
- **일반 지식베이스**: `SKILL.md` + 해당 `references/*.md`를 컨텍스트로 제공

트리거 예: "강도 양형기준", "이 사건 권고 형량", "집행유예 가능성 검토".

## 사용 예시 (Worked Examples)

아래는 **가상 사례**로, 스킬이 산출하는 양형 검토 노트의 형태를 보여줍니다(실제 사건·인물과 무관). 숫자는 해당 `references/*.md`에서 그대로 인용되며, 애매한 지점은 "확인 필요"로 표면화됩니다.

### 예시 1 — 특수강도상해 (금액 아닌 태양 기준 · 특별조정 · 집유)

> 가상 사실: 甲이 야간에 주거 침입, 흉기 휴대, 피해자에게 경미한 상해(치료 2주). 초범, 진지한 반성, 피해액 약 70% 변제, 피해자 처벌불원서 제출.

```
## 양형 검토: 甲 (특수강도상해)

적용: 강도 제2유형(특수강도) / 상해   ─ 형법 제334조 제1항·제335조·제337조
특정강력범죄(누범): 부해당

형량범위(강도 §2-B): 감경 3년~6년 / 기본 4년~7년 / 가중 6년~10년

특별양형인자
  · 감경[경미한 상해 — 치료 2주; 처벌불원 또는 실질적 피해 회복 — 70% 변제·처벌불원서] 2개
  · 가중[없음]                          → 결정 영역: 감경영역
특별 조정: 특별감경만 2개 → 하한 1/2 감경 가능(3년 → 1년6월)
일반양형인자: 감경[진지한 반성; 형사처벌 전력 없음; 상당한 피해 회복]
  → 권고 형량범위: 1년6월 ~ 6년, 선고형 하단부 예상

집행유예
  · 주요참작: 긍정[형사처벌 전력 없음; 처벌불원 또는 실질적 피해 회복] · 부정[위험한 물건의 사용]
  → 종합판단(주요긍정 2 : 부정 1, 2개 이상 차이 아님). 일반긍정 다수 → 집행유예 여지
```

### 예시 2 — 산업안전보건치사 (중대재해 · 경합 유의 표면화)

> 가상 사실: 사업주가 안전조치의무 위반으로 근로자 1명 사망(산업안전보건법 제167조, 제3유형). 초범, 유족과 합의·처벌불원, 재발방지책 시행·진지한 반성.

```
## 양형 검토: 산업안전보건법위반치사

적용: 산업안전보건 제3유형(안전·보건조치의무위반치사)   ─ 산안법 제167조 제1항·제38조
형량범위(과실치사상·산안 §2-B): 감경 6월~1년6월 / 기본 1년~2년6월 / 가중 2년~5년

특별양형인자: 감경[처벌불원 또는 실질적 피해 회복] 1개 → 감경영역 (특별조정 미적용: 2개 미만)
일반양형인자: 감경[진지한 반성; 형사처벌 전력 없음; 위반사항 시정]
  → 권고 형량범위: 감경영역, 선고형 하단부

집행유예: 주요 긍정[처벌불원·실질적 피해 회복] · 부정[사망의 결과 발생]
  → 종합판단. 다만 사망 결과가 주요 부정사유라 실무상 집행유예는 쉽지 않음

⚠️ 유의: 도급 구조라면 도급인(제1유형)과의 경합·다수범죄 처리를 별도 검토(확인 필요)
```

### 예시 3 — 조세포탈(특가법) · 처단형 충돌을 스킬이 표면화

> 가상 사실: 법인 대표가 부가가치세 등 포탈세액 **연 7억 원**(특가법 제8조 제1항 제2호). 초범, 세무조사 단계 수정신고 + 포탈액 약 71% 자진 납부, 진지한 반성, 계획적·조직적 정황 없음.

```
## 양형 검토: 조세포탈(특가법)

적용: 특가법상 조세포탈 제1유형(연 5억~10억 미만)   ─ 특가법 제8조 제1항 제2호
       (법정형: 3년↑ 징역 + 벌금 병과)
형량범위(조세 §2-B): 감경 1년6월~2년6월 / 기본 2년~4년 / 가중 3년~5년

특별양형인자: 감경[포탈한 조세를 상당 부분 납부(71%>2/3); 수정신고] 2개 → 감경영역
  → 특별 조정으로 하한 1/2 감경 가능(1년6월 → 9월)

⚠️ 처단형 우선(공통원칙 02): 특가법 법정형 하한은 3년(징역)이므로, 양형기준 권고
   감경범위와 충돌 → 실제 하한은 법률상 처단형(3년)에 따름. 집행유예는 3년 이하 선고를
   전제하므로 작량감경 없이는 불가 → 처단형·작량감경 검토로 넘어감(확인 필요)
```

> 이처럼 스킬은 형량 숫자를 단정하기보다, 양형기준 권고범위와 **법정형·처단형의 충돌 지점**·**경합 처리 필요**를 표면화하는 데 초점을 둡니다.

## 데이터 신뢰도 — 블라인드 검증 완료

49개 전 범죄군의 형량표를 **독립 추출–대조 방식**(원본 PDF를 별도로 다시 읽어 이 저장소의 표와 셀 단위 대조)으로 검증했습니다. 약 1,100+ 형량 셀에서 **형량 숫자 불일치 0건**. 형종 표기 등 부수 항목 2건만 정정했습니다. 그럼에도 실무 인용 시에는 아래 면책을 반드시 확인하세요.

## ⚠️ 면책 / 유의

- 양형기준은 **권고적 기준**이며 법적 구속력이 없습니다. 최종 선고형은 **법관의 재량**입니다.
- 권고 형량범위가 **법률상 처단형** 범위와 불일치하면 **법률상 처단형이 우선**합니다.
- 양형기준은 **주기적으로 개정**됩니다. 이 저장소는 **2026. 7. 1. 시행본** 기준입니다. 일부 파일 헤더의 "밑줄 개정조항" 경고 참조. **현행 여부는 반드시 양형위원회 공식본으로 확인하세요**: <https://sc.scourt.go.kr>
- 이 스킬의 산출물은 **실무 참고용 검토 노트**이며, 법률자문·의견서·준비서면 등 정식 법률문서를 대체하지 않습니다.
- 본 저장소는 양형위원회의 공식 산출물이 아니며, 위원회와 무관합니다.
