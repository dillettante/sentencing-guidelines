# 양형기준 적용 스킬 (Korean Sentencing Guidelines Skill)

대한민국 **양형위원회 『2026 양형기준』(2026. 7. 1. 시행)** 을 조회·적용해, 특정 범죄사실에 대한 **권고 형량범위·집행유예 판단·양형 검토 노트**를 산출하는 에이전트 스킬입니다. 양형기준이 설정된 **49개 전 범죄군**의 형량범위표·양형인자표·집행유예표·다수범죄 특칙이 구조화되어 있습니다.

요약 도구가 아니라 판단 파이프라인입니다:
`사실관계 → 범죄군·유형 특정 → 형량범위 조회 → 양형인자 평가 → 집행유예 판단 → 검토 노트`

---

## 구성

```
sentencing-guidelines/
├── SKILL.md            # 공통 엔진: 적용 5단계 + 평가·공통원칙 + 다수범죄 + 산출 포맷 + 정확성 게이트 + 49개 색인
└── references/         # 49개 범죄군 데이터 (범죄군별 1파일)
    ├── gangdo.md       # 강도
    ├── sagi.md         # 사기
    ├── hoengnyeong.md  # 횡령·배임
    └── ... (총 49개)
```

각 reference 파일: **유형정의(구성요건·적용법조) · 형량범위표(감경/기본/가중) · 양형인자표(특별·일반 × 감경·가중 × 행위·행위자) · 인자정의 · 집행유예 기준**. 전 범죄군에 공통인 평가원칙·공통원칙·표준 다수범죄 처리기준은 중복을 피해 `SKILL.md`에만 두었고, 범죄군 고유 특칙(예: 이득액 합산)만 각 파일에 있습니다.

## 사용법

에이전트 스킬 디렉터리에 배치하면 트리거 시 자동 로드됩니다.
- **Claude Code**: `~/.claude/skills/sentencing-guidelines/`
- **GitHub Copilot CLI / Amp** 등: 각 에이전트의 skills 루트에 배치
- **일반 지식베이스**: `SKILL.md` + 해당 `references/*.md`를 컨텍스트로 제공

트리거 예: "강도 양형기준", "이 사건 권고 형량", "집행유예 가능성 검토".

## 데이터 신뢰도 — 블라인드 검증 완료

49개 전 범죄군의 형량표를 **독립 추출–대조 방식**(원본 PDF를 별도로 다시 읽어 이 저장소의 표와 셀 단위 대조)으로 검증했습니다. 약 1,100+ 형량 셀에서 **형량 숫자 불일치 0건**. 형종 표기 등 부수 항목 2건만 정정했습니다. 그럼에도 실무 인용 시에는 아래 면책을 반드시 확인하세요.

## ⚠️ 면책 / 유의

- 양형기준은 **권고적 기준**이며 법적 구속력이 없습니다. 최종 선고형은 **법관의 재량**입니다.
- 권고 형량범위가 **법률상 처단형** 범위와 불일치하면 **법률상 처단형이 우선**합니다.
- 양형기준은 **주기적으로 개정**됩니다. 이 저장소는 **2026. 7. 1. 시행본** 기준입니다. 일부 파일 헤더의 "밑줄 개정조항" 경고 참조. **현행 여부는 반드시 양형위원회 공식본으로 확인하세요**: <https://sc.scourt.go.kr>
- 이 스킬의 산출물은 **실무 참고용 검토 노트**이며, 법률자문·의견서·준비서면 등 정식 법률문서를 대체하지 않습니다.
- 본 저장소는 양형위원회의 공식 산출물이 아니며, 위원회와 무관합니다.
